    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1><a id="user-content-dlw" class="anchor" href="#dlw" aria-hidden="true"><span class="octicon octicon-link"></span></a>DLW</h1>

<p>This is another Digital Light Wand software based on the ideas of Michael Ross (<a href="http://mrossphoto.com/wordpress32/dlwblog/">http://mrossphoto.com/wordpress32/dlwblog/</a>)</p>

<p>It is based on 144 LED Adafruit stips (<a href="https://www.adafruit.com/products/1507">https://www.adafruit.com/products/1507</a>) with a Arduino Mega, display shield and SD.</p>

<p>This DLW software is coded completely from scratch, preset to 288 Leds in two strips. It reads BMP files from SD and has an option for various delays.
There is also the possibility to add a Buzzer. Future planned features include a remote for a DSLR.</p>

<p>I'm not at all an experienced coder, so the thing looks quite messy... eventually I may (or may not) clean it up a bit.</p>
